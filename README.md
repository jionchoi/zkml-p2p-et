# ZKML system comparitive analysis

Comparative analysis of different open-source ZKML frameworks/systems through benchmarking and experiments.


# Related Work

In 2024, `zkonduit`, the team of EZKL, benchmarked three frameworks `EZKL`, `Orion`, and `RISC0` in terms of `proving time` and `memory usage` across four commonly used ML models: `Linear Regression`, ``.
They posted their benchmarking results in this [blog](https://blog.ezkl.xyz/post/benchmarks/).
The original benchmark project is hosted in this [GitHub repo](https://github.com/zkonduit/zkml-framework-benchmarks/tree/main).


Another [similar work](https://github.com/Guy1m0/ZKML-Benchmark/tree/main) was done by `Guy1m0`. In this project, four frameworks, `circomlib`, `ezkl`, `opML`, and `zkml` by Daniel Kang, have been bencharked. In this benchmarking, accuracy was considered in addition to resources.
